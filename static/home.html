CAPABILITIES

Built for Serious Traders

Production-grade features designed from real trading experience, not hype.

Multi-Exchange

Execute simultaneously on Binance, OKX, Bybit, Bitget, Gate.io, and Coinbase. Unified API with automatic symbol resolution and contract size handling.

Multi-Take-Profit

Up to 4 TP levels with quantity split. Each level tracked independently with order verification and automatic re-placement on exchange cancellation.

Smart Trailing Stop

5 trailing modes: none, auto (AI-selected), moving, breakeven-on-TP1, step-trailing. Re-evaluated on limit order fills for changed market conditions.

24/7 Position Monitor

Continuous reconciliation with exchange data. Ghost position detection with API-failure safeguards. Protective order verification and automatic re-placement.

SIGNAL PROCESSING

Reliable Signal Ingestion

Secure webhooks with replay protection, IP rate limiting, and daily trade caps ensure your signals are processed exactly once without over-trading.

  • HMAC signature verification for signal authenticity
  • IP-based rate limiting (60 req/min) prevents abuse
  • Fingerprint-based replay protection against duplicates
  • Per-user daily trade limits with database persistence
  • Emergency stop, pause, and read-only trading modes
